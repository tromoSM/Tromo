Create a CSS File with Notepad

Step 1: Open Notepad

Step 2: Type your CSS like this:

body {
  background-color: aliceblue;
  font-family: Arial, sans-serif;
  overflow: hidden; /* This disables scrolling */
}

Step 3: Save the file as style.css

Step 4: Link it in your HTML using:

<link rel="stylesheet" href="style.css">
